# Projetos

Esta pasta contém os projetos desenvolvidos durante o curso.

## Estrutura recomendada:

Cada projeto deve ter sua própria pasta com:
- README.md explicando o projeto
- Código fonte
- Documentação
- Instruções de execução

## Exemplo:
```
projetos/
├── projeto-web/
│   ├── README.md
│   ├── src/
│   └── docs/
└── projeto-mobile/
    ├── README.md
    └── app/
```
