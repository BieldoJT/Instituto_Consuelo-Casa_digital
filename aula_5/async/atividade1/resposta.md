# Qual a diferença entre PUT e PATCH?

o PUT é usado para alterar todo o campo da requisição, então se for necessário fazer um alteração num campo espefico , deve repitir as alterações. Ja o PATCH é usado para alterar campos, sem a necessidade de colocar todos os campos, é possivel fazer alterações do campo unitario

# Por que usar um endpoint específico para atualizar estoque?

Para ter mais controle e garantia que não vai ter alteração das informações do produto. Então tem mais segurança e garantia de que so o estoque vai ser alterado
